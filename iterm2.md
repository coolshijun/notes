
Window / Pane

⌘ + T         New Tab
⌘ + D         New vertical pane
⌘ + SHIFT + D New horizontal pane

⌘ + [         Previous pane
⌘ + ]         Next pane

Fullscreen

⌘ + SHIFT + Enter Fullscreen

⌘ + SHIFT + H Paste history

⌘ + ⌥ + B   Instant reply, ESC to exit

⌘ + ;   Autocomplete

⌘ + F   Search content / Tab / SHIFT + Tab to select 

⌥ + Enter to Paste select content
