# Useful Shortcut

Double "⇧" -> Search Everywhere

⇧ + ⌘ + A -> Search Actions

⌘ + O  -> Search Classes

⇧ + ⌘ + O -> Search Files

⌥ + ⌘ + O - > Search Symbols

⌃ + TAB -> Switcher of Panes or Open files

⌃ + ` -> Switch Scheme / View Mode

⌘ + ↑ -> Navigation bar

⌥ + F12 -> Toggle Terminal

⌘ + 1 - > Toggle Project pane

⌥ + ←  -> Move one word back

⌥ + →  -> Move one word forward

⌘ + ←  -> Move back to start

⌘ + →  -> Move forward to end
